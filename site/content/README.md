# Content Guide

This folder contains owner-manageable content sources for the KBFA site.

## Current V1 Structure

- `settings/`: site-level bilingual settings
- `activities/`: featured activity data
- `sponsorship/projects/`: project sponsorship entries
- `popups/`: popup-ready notice data

## Editing Principles

- Keep Korean and English content separate
- Do not publish placeholder facts as confirmed public information
- When contact details, bank account details, or image approvals are not final, keep them blank or clearly pending

## Future Direction

Markdown templates are included for future operational expansion even if the current V1 app still reads TypeScript and JSON seed data first.
